VERIFICATIONS FRONTEND - SKILLBRIDGE
====================================

Dossier a ouvrir dans le terminal
---------------------------------
Commande :
cd "C:\Users\elcor\Desktop\exam titre pro\skillbridge\frontend"

A quoi ca sert :
Toutes les commandes frontend doivent etre lancees depuis ce dossier.


1. Verifier que le backend est lance
------------------------------------
URL navigateur ou Postman :
http://localhost:5000/api/health

Resultat attendu :
{
  "status": "ok",
  "message": "SkillBridge API is running"
}

A quoi ca sert :
Le frontend appelle l'API backend. Si le backend ne tourne pas, login, dashboard, groupes, sessions et admin ne fonctionneront pas.


2. Verifier le fichier d'environnement frontend
-----------------------------------------------
Fichier :
frontend/.env

Valeur attendue :
VITE_API_URL=http://localhost:5000/api

A quoi ca sert :
Indique a Axios l'adresse de l'API backend.


3. Installer les dependances frontend
-------------------------------------
Commande :
npm install

Resultat attendu :
Installation terminee sans erreur bloquante.

A quoi ca sert :
Installe React, Vite, Axios, React Router, React Hook Form, Zod, Tailwind et Lucide.


4. Lancer le frontend en developpement
--------------------------------------
Commande :
npm run dev

URL attendue :
http://localhost:5173

A quoi ca sert :
Demarre l'application React avec Vite.


5. Verifier la compilation frontend
-----------------------------------
Commande :
npm run build

Resultat attendu :
Build termine sans erreur TypeScript.

A quoi ca sert :
Verifie que le frontend est compilable pour la production.


6. Verifier le lint frontend
----------------------------
Commande :
npm run lint

Resultat attendu :
Aucune erreur ESLint.

A quoi ca sert :
Verifie la qualite du code, les hooks React, les imports de types et les erreurs courantes.
